# TaskMaster Pro - Dynamic Task Management System

## Overview
A sophisticated web application demonstrating advanced frontend and backend integration using HTML, CSS, JavaScript, and PHP. This project showcases modern web development practices suitable for a Master's degree assignment.

## Features
- **Dynamic Task Management**: Create, read, update, and delete tasks
- **Real-time Statistics**: Live updates of task metrics
- **Advanced Filtering**: Filter tasks by status (all, pending, completed)
- **Priority System**: Categorize tasks by priority levels
- **Responsive Design**: Mobile-first responsive layout
- **RESTful API**: Clean PHP backend with proper HTTP methods
- **Modern UI/UX**: Professional gradient design with smooth animations

## Technical Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP with RESTful API
- **Data Storage**: JSON file-based storage (simulates database)
- **Styling**: CSS Grid, Flexbox, and modern design principles

## Architecture Highlights
1. **MVC-inspired Pattern**: Separation of concerns between presentation and data layers
2. **Asynchronous Operations**: AJAX requests for seamless user experience
3. **RESTful Design**: Proper use of HTTP methods (GET, POST, PUT, DELETE)
4. **Security Measures**: Input sanitization and CORS headers
5. **Error Handling**: Comprehensive error handling and user feedback

## Installation
1. Place all files in a web server with PHP support
2. Ensure write permissions for the `tasks.json` file
3. Access `index.php` through a web browser

## API Endpoints
- `GET tasks.php` - Retrieve all tasks
- `POST tasks.php` - Create new task
- `PUT tasks.php` - Update task completion status
- `DELETE tasks.php` - Delete task
